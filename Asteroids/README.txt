Spelet handlar om:
Ett klassiskt asteroidspel där spelaren styr ett rymdskepp och ska överleva genom att undvika och skjuta ner asteroider. Spelet går ut på att samla poäng genom att förstöra asteroider samtidigt som man försöker hålla sig vid liv.

Funktioner:

Rörelse:
Spelaren kan styra rymdskeppet genom att rotera vänster och höger samt röra sig framåt. Rymdskeppet kan också skjuta projektiler för att förstöra asteroider.

Asteroider:
Asteroider spawnar kontinuerligt och rör sig slumpmässigt. När en stor asteroid träffas delas den upp i två mindre delar. Mindre asteroider försvinner helt när de förstörs.

Livssystem:
Spelaren börjar med tre liv, som visas som hjärtsymboler på skärmen. Vid kollision med en asteroid förlorar spelaren ett liv. När alla liv är slut visas en Game Over-skärm.

Poängsystem:
Spelaren får poäng baserat på storleken på de asteroider som förstörs: stora asteroider ger minst poäng, medan mindre ger mer. Poängen visas i spelet.

Mål:
Undvik att förlora alla liv och försök samla så många poäng som möjligt. Spela om genom att trycka på Enter efter Game Over.
